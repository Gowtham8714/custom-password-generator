# Memorable Password Generator 🔐

A lightweight Python script that creates highly secure, unpredictable passwords using your memorable details.

## ✨ Features
Dynamic Slicing: Extracts random 3-5 letter chunks from your inputs so the pattern is never predictable.
Smart Date Parsing: Automatically converts text dates (e.g., "2nd April") into clean numbers ("04").
Full Randomization: Completely shuffles the name slices, city slices, dates, and symbols.
Privacy-First: Prompts users to add a mental "secret pin" at the end, ensuring absolute zero-knowledge security.
